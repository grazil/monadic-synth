module Operator: sig .. end

Operators a stream generators. They consist of two combined monads: one for the state and one for streams (as in Stream). They should replace Stream if they prove practicable as one could forsee compiling those.

type 'a typ =
| Float : float typ
| Pair : ('a0 typ * 'b typ) -> ('a0 * 'b) typ 
type types =
| Type : 'a typ -> types 
type 'a reference = 'a typ * int 
type 'a value =
| Unit : unit value
| Float : float -> float value
| Ref : 'a0 reference -> 'a0 reference value
| Get : 'a1 reference -> 'a1 value
| Set : 'a2 reference * 'a2 value -> unit value
